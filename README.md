## Introduction

The Notepad App is a web application developed using Next.js and React. It allows users to write, edit, and save notes in a user-friendly interface. The app supports opening and saving notes as text files.

## Features

- Create and edit notes using a simple interface.
- Open existing notes from your local system.
- Save notes as text files.
- User-friendly and responsive design.
