## Introduction

The Notepad App is a web application developed using Next.js and React. It allows users to write, edit, and save notes in a user-friendly interface. The app supports opening and saving notes as text files.

## Features

- Create and edit notes using a simple interface.
- Open existing notes from your local system.
- Save notes as text files.
- User-friendly and responsive design.

## Getting Started

### Installation

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd notepad
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Usage

1. Start the development server:
   ```bash
   npm run dev
   ```

2. Open your web browser and navigate to [http://localhost:3000](http://localhost:3000) to access the Notepad App.

3. Use the buttons to open and save notes, and type in the textarea to write and edit notes.
